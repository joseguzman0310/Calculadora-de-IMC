# Calculadora-de-IMC
Sitio web de proyecto personal como campaña para la facultad de medicina de la Universidad Diego Portales que consiste en el calculo del IMC y el estado nutricional de los
Alumnos que estudian en las distintas facultades de la Universidad Diego Portales en las sedes de Santiago Centro y Huechuraba
El sitio web esta realizado con CSS, HTML Y Javascript
